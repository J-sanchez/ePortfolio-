Description

For my ePortfolio, I selected the SNHU Travel: Niche Vacation Booking System project from CS 250: Software Development Lifecycle. The original application was a simple Java desktop app showing five vacation destinations in a slideshow format.

The enhanced version transformed the Java app into a modern web tool using Vite, React, Tailwind CSS, Express.js, and the Amadeus API. Key features include:

  • Flight Search: Search for departure and return flights using the Amadeus Flights API.
  • Hotel Search: Search for hotels in specific cities with links to Expedia for booking.
  • Experience Search: Find points of interest in cities using the Amadeus POIS API.
  • User-Friendly Interface: Clean, responsive pages for easy navigation.
  • Four Pages: Home, Flights, Hotels, and Experiences with intuitive design.

Key Skills Demonstrated:

  • API Integration: Integrated real-time data from the Amadeus API for flights, hotels, and experiences.
  • React & Web Development: Built a modern web app with Vite, React, and Express.js.
  • User Interface Design: Focused on providing an intuitive and visually appealing user experience.
  • Problem-Solving: Overcame challenges in working with complex APIs and integrating dynamic data.

This project showcases my ability to take a simple idea and turn it into a fully functioning web application with modern technologies. If revisited, I'd add features like user login and more advanced booking capabilities.
