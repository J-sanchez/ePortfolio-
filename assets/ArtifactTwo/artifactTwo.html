Description

For my ePortfolio, I selected the Binary Search Tree (BST) visualization project from CS 300: Data Structures and Algorithms. The original version was a straightforward implementation of tree data structures and algorithms, such as insertion, deletion, and traversal. It worked but lacked any visual or interactive elements to enhance understanding.

The enhanced version turned the BST into a web tool using React, react-d3-tree, and Tailwind CSS. Key features include:

  • Random Tree Generation: Generates a random BST for users to interact with.
  • Traversal Visualization: Highlights nodes in real-time for pre-order, in-order, and post-order traversals.
  • Interactive UI: Simple, clean controls to generate new trees and start traversal.
  • About Page: Explains how tree traversal algorithms work.
  • Responsive Design: Tailwind CSS ensures the app looks good on all devices.

Key Skills Demonstrated:

  • React & react-d3-tree Integration: Created an interactive UI with React and visualized the tree structure with react-d3-tree.
  • Web Development Practices: Built with modern web technologies like React and Tailwind CSS.
  • Data Structure Understanding: Solidified my grasp of BSTs and their traversal algorithms.
  • UI/UX Design: Focused on creating a user-friendly interface, ideal for educational purposes.

This project allowed me to combine my understanding of data structures with modern web tools to create an interactive experience. If revisited, I would explore adding more tree structures and traversal methods to further enhance the app.
